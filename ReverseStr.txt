Key constraints to keep in mind :
-Input is array of characters, not a string
-Must modify in-place
-O(1) extra space
-Length can be large â†’ efficiency matters

1. Index-based swapping approach ->

class Solution {
    public void reverseString(char[] s) {
        for(int i =0; i< s.length/2; i++){
            char temp = s[i];
            s[i] = s[s.length-1-i];
            s[s.length-1-i] = temp;
        }
    }
}

2. Two-Pointer Approach -> (optimal solution)

class Solution {
    public void reverseString(char[] s) {
        //using two pointer approach
        int left = 0;
        int right = s.length -1;

        while(left < right){
            char temp = s[left];
            s[left] = s[right];
            s[right] = temp;

            left++;
            right--;
        }
    }
}
